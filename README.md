## 📋 Overview

This repository documents a structured cyber risk assessment performed for a cloud-hosted Learning Management System (LMS).

The project demonstrates the development and maintenance of a cyber risk assessment workbook including asset identification, threat analysis, vulnerability assessment, risk evaluation, security controls assessment, treatment planning, and risk register management.

---

## 🎯 Assessment Objectives

The assessment was conducted to:

* Identify critical LMS assets
* Identify threats affecting the LMS environment
* Document vulnerabilities impacting asset security
* Assess risk using impact and likelihood scoring
* Review security controls
* Develop treatment actions to reduce risk exposure
* Maintain a centralized risk register

---

## 🛠️ Assessment Components

| Component | Purpose |
| --- | --- |
| Asset Register | Documents critical business assets |
| Threat Register | Identifies potential threats |
| Vulnerability Register | Records identified weaknesses |
| Risk Assessment Register | Calculates and evaluates risk |
| Risk Matrix | Visualizes risk severity |
| Security Controls Register | Documents security controls |
| Risk Treatment Plan | Defines mitigation activities |
| Risk Register Summary | Tracks overall risk posture |

---

## 🏢 Assessment Scope

The assessment included:

* LMS Web Application
* Database Server
* Student User Accounts
* Administrative Accounts
* Cloud Hosting Environment
* Backup Server

---

## ⚠️ Threats Evaluated

* Credential Stuffing
* SQL Injection
* Distributed Denial of Service (DDoS)
* Privilege Misuse
* Ransomware

---

## 🔎 Vulnerabilities Assessed

* No MFA enabled for student accounts
* Inadequate input validation
* No Web Application Firewall (WAF)
* Excessive administrative privileges
* Backups not regularly tested

---
